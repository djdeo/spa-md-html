# Title from readme.md

Add `.md` file and it display `.html` automatically

```
git clone this repo
```

Run `index.html` file from `localhost`

## TOC

Write in markdown format, create table of content automatically 

## SEO

This is kind of like single page application, only `index.html` file will be indexed

Want to index all the pages, use static website instead

## Add new page

create a page in the `docs` folder and include it in `sidebar.md`

## Usage

- Product manual
- Software Tutorial
- Display open source project
- Other SPA site
